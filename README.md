# NorthwindProject
This is a .NET Core Back-End project with N-Tier Architecture, SOLID, OOP and AOP techniques.

Projede büyük ölçüde, Değerli Engin Demiroğ Hocamızın sunmuş olduğu içeriklerden ve kendisinin yakın zamanda gerçekleştirdiği ücretsiz kampından faydalandığımı belirtmek isterim. Bize böyle değerli bilgiler paylaştığı için kendisine en içten dileklerimle teşekkürlerimi sunuyorum.

Bu proje şu anda geliştirilme aşamasındadır. Projeye yakın zamanda yeni özellikler eklenecek, front-end tarafı da eklenecek ve metodlar asenkron hale getirilecektir.
