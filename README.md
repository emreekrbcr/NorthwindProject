# NorthwindProject
This is a .NET Core Back-End project with N-Tier architecture and SOLID, OOP and AOP techniques.

Bu proje geliştirilme aşamasındadır. Projeye yakın zamanda yeni özellikler eklenecek, front-end tarafı da eklenecek ve metodlar asenkron hale getirilecektir.

Projede büyük ölçüde, Değerli Engin Demiroğ Hocamızın sunmuş olduğu içeriklerden ve ücretsiz kampından faydalandığımı belirtmek isterim. Bize böyle değerli bilgiler paylaştığı için kendisine en içten dileklerimle teşekkürlerimi sunuyorum.
